# homebrew-tap

Homebrew tap for [seunggabi](https://github.com/seunggabi) projects.

## Formulas

| Formula | Description |
|---------|-------------|
| [mac-ops](https://github.com/seunggabi/mac-ops) | macOS system optimization CLI tool |
| [claude-dashboard](https://github.com/seunggabi/claude-dashboard) | k9s-style TUI for managing Claude Code sessions |

## Install

```bash
brew tap seunggabi/tap
```

### mac-ops

```bash
brew install seunggabi/tap/mac-ops
```

### claude-dashboard

```bash
brew install seunggabi/tap/claude-dashboard
```

## Upgrade

```bash
brew upgrade mac-ops
brew upgrade claude-dashboard
```

## Uninstall

```bash
brew uninstall mac-ops
brew uninstall claude-dashboard
brew untap seunggabi/tap
```
